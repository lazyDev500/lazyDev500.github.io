## 레시피

국물 두루치기 만들기

환절기라 감기에 걸린거 같다.
목이 칼칼하고 기침과 제체기가 계속 나온다.
뜨겁고 얼큰한 국물과 몸보신을 위한 고기가 먹고 싶다.
그럴 땐 국물 두루치기지

난이도 ✭✭✭

감기가 달아 나는 얼큰한 국물 두루치기

재료

재료

  1. 돼지고기 / 150g
  2. 양파 / 1/4개
  3. 대파 / 반대
  4. 청양고추 / 1~2개
  5. 다진 마늘 / 1sp

양념/조미료

  1. 고추장 / 2sp
  2. 고추가루 / 1sp
  3. 진간장 / 1sp
  4. 설탕 / 1sp
  5. 맛술 / 1sp
  6. 참기름 / 1sp
  7. 후추 / 적당히

레시피

  1. 고추장 2sp, 고추가루 1sp, 진간장 1sp, 설탕 1sp, 맛술 1sp, 참기름 1sp, 후추 조금을 모두 섞어 양념장을 만든다.

  2. 양파는 채썰고 대파와 고추는 적당한 크기로 썰어 준비한다.

  3. 돼지고기와 다진 마늘을 냄비에서 중불로 볶아 준다.

  4. 돼지고기가 익기 시작하면 양파와 고추, 양념장을 넣고 물 200ml를 부어준 후 강불로 끓인다.

    • 이때 적당히 익은 김치를 함께 넣어주면 김치 두루치기가 된다

  5. 국물이 적당히 줄어 들면 참기름 조금과 대파 초록 부분을 넣고 마무리 한다.

  6. 맛잇게 먹는다.
